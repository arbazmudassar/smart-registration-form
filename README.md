# 📝 Smart Registration Form

A responsive and interactive registration form designed with a focus on user experience and real-time validation. This project demonstrates how modern web forms handle user input, validation, and error feedback effectively.

---

## 📌 Overview

Forms are a critical part of almost every web application. This project simulates a real-world registration system where users can input their details, and the system validates the data before submission.

The main goal of this project is to improve usability by providing instant feedback and preventing incorrect data entry.

---

## 🚀 Features

* 🧾 User-friendly registration form
* ⚡ Real-time input validation
* ❌ Error messages for invalid inputs
* 📧 Email format validation using Regex
* 🔒 Password strength and confirmation check
* 📱 Fully responsive design
* 🎨 Clean UI with Bootstrap

## 🧠 Validation Logic

The form uses JavaScript and regular expressions to validate:

* Email format (e.g., [user@example.com](mailto:user@example.com))
* Password strength
* Required fields
* Matching passwords
